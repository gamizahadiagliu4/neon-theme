# Changelog

All notable changes to the "Neon Cyberpunk Theme" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-03

### Added
- Initial release of Neon Cyberpunk Theme
- Main theme variant with full intensity neon colors
- Soft theme variant with muted neon colors for comfortable viewing
- Complete color scheme based on cyberpunk aesthetics
- Support for all major programming languages:
  - JavaScript/TypeScript
  - Python
  - HTML/CSS
  - JSON
  - Markdown
  - And many more
- Comprehensive syntax highlighting with semantic support
- Optimized terminal colors with 16 ANSI color support
- High contrast design for excellent readability
- Eye-friendly color palette for night coding sessions
- Full VS Code UI theming including:
  - Activity Bar
  - Side Bar
  - Editor Groups and Tabs
  - Status Bar
  - Panels
  - Settings UI
  - Git decorations
  - Error/Warning indicators
- Bracket pair colorization support
- Comprehensive documentation and installation guide

### Color Palette
- Background Primary: #0a0a0f (deep dark blue)
- Background Secondary: #0d0d14 (layering support)
- Background Tertiary: #06060a (maximum depth)
- Text Primary: #e0e0ff (light blue with purple tint)
- Text Secondary: #a0a0d0 (muted blue-gray)
- Accent Cyan: #00ffff (primary neon accent)
- Accent Magenta: #ff00ff (highlighting)
- Accent Yellow: #ffff00 (functions/warnings)
- Accent Violet: #8a2be2 (secondary accent)
- Accent Green: #00ffaa (strings/success)
